Advanced AI functionalities that we have used in this project.

Using advanced AI techniques


Transfer leraning

Leveraging a pre-trained machine learning model called MediaPipe for extracting meaningful features from raw video data frames to create a feature representation that is more informative and easier to use for getting the keypointers of the body when signing.


Deep learning techniques

Using RNN (LSTM) as an advanced AI technique for training a model for detecing sign language.

Data processing for raw dataset


Special preprocessing of video data

We Download and crop videos from dataset based on specified time intervals and then extract keypoints from frames. The keypoints are saved in NumPy arrays organized by action labels. The data is retrieved from a SQLite database, and the final processed data is stored in specific directories.

Main functionalities of the system


Live translation

The system is able to do predictions for the live signed word and display the text on the stream.


Translate recorded vidoes

The system is able to translate the words signed in a recorded video and provide a downloadable transcript.
